# Spacepress

A simplistic CMS blog.

## Dependencies
- Ruby v3.0.1
- Rails v6.1.4

## Installation (Local)

clone the repository: 

```
$ git clone https://github.com/adamtlee/spacepress.git
```

Install the Gem dependencies: 

```
$ bundle install
```

Run the existing migrations: 

```
$ bundle exec rake db:migrate
```
